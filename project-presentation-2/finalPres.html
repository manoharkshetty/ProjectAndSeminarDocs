Abstract

    Switch-Hit is a fantasy cricket application with the unmatched social experience. It will provide a platform that help friends to connect and make their own team with players they like and have fun with the friends. Users can view who is topper of league their batting, bowling and fielding points. Switch-Hit provides features to post, comment, Hi-Fi the post, schedule and players performance information

INTRODUCTION

  • Cricket is most loved sport in more than 15 countries.

  • The popularaity of the game is so huge, which made it to be available in many formats like Test Cricket, One day International and T-20.

  • So there is a need for a platform that help friends to connect and make their own team with players they like and have fun with the friends.

  • So we are planning to make fantasy cricket application with the unmatched social experience.

Proposed System

  • Fantasy cricket application with the unmatched social experience where they can hang out with friends talk about cricket, post their comments, like the other's comments, etc.

  • Mobile applications enable users to change the team while travelling or working which increase the user engagement.

  • So Switch-Hit is the one such attempt.

DESIGN

  • User authentication is done through Facebook authentication which allows to use users social information
  • Facebook credentials are used as key to fetch user information.
  • The data is transferred in the form of json data which is light weight
  • The android communicates to Switch-hit server through API calls

Implementation

  • To use the facebook authentication we have to register with the facebook by providing them the application's fingerprint
  • once we get the accesstoken and facebookid we can get the user object from switch-Hit server
  • the data from the server is accessed using the light weight rest apis
  • switch-hit uses loopj's asynchronous http client
  • the response json is converted into java object using the google library "gson"
